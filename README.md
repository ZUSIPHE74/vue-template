# Steps to Install and create a Vue.js app with Vuex
1. npm create vue@latest
2. named our project - "vue-frontend"
3. features - choose JSX and Router
4. experimental features - Oxlint
5. We said no to skipping template code because want to see the views,components, etc
6. cd "vue-frontend"
7. npm install
8. npm run dev
9. ctrl c - stop server from running
10. npm install vuex@latest
